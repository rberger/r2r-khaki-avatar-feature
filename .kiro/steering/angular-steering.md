# Technology Stack

## Core Dependencies

- **Angular**: Version 20.3.9
- **TypeScript**: Version 5.9.3
- **AWS SDK**: `@aws-sdk` - AWS service integration

## Key Technologies

- **TypeScript**: Strict mode enabled with comprehensive type checking
- **RxJS**: Version 7.x for reactive programming
- **Rollup**: For bundling packages
- **esbuild**: Fast JavaScript bundler
- **Prettier**: Code formatting (enforced by CI)
- **TSLint**: Linting (following Google TypeScript Style Guide)
- **Jasmine/Karma**: Testing framework
- **Protractor**: E2E testing
- **Cypress**: E2E testing for DevTools

## Styling
- **Sass** - CSS preprocessor (`.sass` syntax, not `.scss`)

## TypeScript Configuration

- **Target**: ES2022
- **Module**: ESNext with bundler resolution
- **Strict mode**: Enabled with all strict flags
- **Decorators**: Experimental decorators enabled
- **Source maps**: Enabled with inline sources
- Line wrap at **100 characters**
